## TeensySynth - a polyphonic MIDI synthesizer for Teensy 3.x

### Features
* 8-voice polyphony
* 6 waveforms: sine, square, triangle, sawtooth, reverse sawtooth, pulse
* filter and amplitude per voice
* MONO and POLY mode

### Supported MIDI controls
* Channel volume
* Channel pan
* Sustain pedal
